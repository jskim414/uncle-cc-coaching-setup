# 클코형 1:1 클로드 코드 과외 참조자료

비개발자 대표님·실무자를 위한 **클로드 코드(Claude Code) 1:1 과외** 실습 자료 저장소입니다.
설치·기본 사용법부터 리서치·토이 프로젝트 실전까지, 과외에서 다루는 모든 참조 문서를 단계별로 모아두었습니다.

---

## 📂 목차

### 01. 기초 문서 — 설치부터 심화 개념까지

처음 클로드 코드를 설치하고, 기본 원리·MCP·훅 같은 핵심 개념까지 익히는 4단계 자료입니다.

| 단계 | 파일 | 내용 | 소요 |
|------|------|------|------|
| STEP 1 | [완전 설치·세팅 가이드](01-기초-문서/STEP1_완전-설치-세팅-가이드.md) | 처음부터 순서대로 설치 | 40~60분 |
| STEP 2 | [설치 체크리스트](01-기초-문서/STEP2_설치-체크리스트.md) | 이미 뭐가 깔렸는지 5분 진단 | 5분 |
| STEP 3 | [기초 상식](01-기초-문서/STEP3_기초-상식.md) | 개념 지도 (3원리·CLAUDE.md·MCP·훅) | 20분 |
| STEP 4 | [심화 상식](01-기초-문서/STEP4_심화-상식.md) | 풍부하게 쓰기 8주제 (API·Cron·Git 등) | 25분 |

## 🎯 대상

- 40대 비개발자 대표님 / 실무자
- Windows 10·11 기본, macOS 병기
- Claude Code v2.1.x 기준 (2026-04)

## 🚀 사용 순서

1. **처음이라면** → `01-기초-문서/STEP1`부터 순서대로
2. **이미 일부 설치되어 있다면** → `01-기초-문서/STEP2`로 5분 진단
3. **설치가 끝났다면** → `STEP3 → STEP4`로 개념 익히기
4. **업무에 적용** → `02-리서치-프롬프트`로 실제 리서치 프롬프트 활용
5. **내 아이디어 구현** → `03-토이-프로젝트`의 Plan 템플릿으로 시작
