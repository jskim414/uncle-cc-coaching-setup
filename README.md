| 단계 | 파일 | 내용 | 소요 |
|------|------|------|------|
| STEP 1 | [완전 설치·세팅 가이드](기초-문서/STEP1_완전-설치-세팅-가이드.md) | 처음부터 순서대로 설치 | 40~60분 |

## 🎯 대상

- 40대 비개발자 대표님 / Windows 10·11 기본, macOS 병기

## 🚀 시작 방법

1. 처음이라면 **STEP 1**부터 순서대로

3. 설치가 끝났다면 **STEP 3 → STEP 4** 로 개념 익히기
