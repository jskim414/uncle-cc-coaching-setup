# 엑셀 파일 위젯

## 1. 사용자
 - hwpx, docx 형태의 문서 작업을 주로 하는 직장인
 - Windows 11 환경

## 2. 과업
 - 특정 스프레드시트(.xlsx) 파일에 있는 항목을 복사해서 작성 중인 문서에 붙여넣기
 - 개별 셀 하나 클릭 복사, 또는 드래그해서 영역 복사
 - 데이터가 많을 때 검색/필터로 원하는 항목 빠르게 찾기

## 3. 완료 기준
 - 프로그램 창을 왔다갔다하지 않고, 엑셀 데이터가 항상 위젯처럼 플로팅
 - 셀 클릭만으로 바로 클립보드에 복사 (자동복사 모드)
 - 선택 후 Ctrl+C로 복사하는 모드도 전환 가능
 - 투명도 조절로 문서 작업 방해 최소화
 - 고정(Always on Top) on/off 전환 가능

## 4. 금지 / 예외
 - .xls(구형 엑셀) 지원 불필요, .xlsx만
 - 엑셀 데이터 편집 기능 불필요 (읽기 전용)
 - 여러 파일 동시 열기 불필요 (한 번에 1개)

## 5. 맥락
 - 예: 전문가별 계좌번호, 은행명, 주민등록번호를 포트폴리오 문서에 반복 붙여넣기
 - 어떤 프로그램(한글, 워드 등) 위에든 가장 위에 떠 있는 플로팅 위젯
 - 대용량 엑셀(수천 행)에서도 버벅이지 않아야 함
 - 비개발자도 쓸 수 있게 .exe 단일 파일 배포 가능해야 함
