# 웹에서 편하게 사용하는 마크다운 에디터 

## 1. 사용자
 - 나 자신 
 - 클로드코드 업무 OS로 적극 활용하고 있음. 비개발자이지만 다양한 문서,지식 업무에 클로드코드를 기본으로 사용
 - 그러다보니 마크다운 문서로 소통을 많이하는 상황
 - 현재 사용하고 있는 IDE는 ZED, Antigravity 등. 하지만 단순 마크다운 문서 편집, 열기할 때 IDE를 활용하기에 뭔가 아까움 (폴더 설정 리셋 등)

## 2. 과업
 - 온라인에서 마크다운을 읽고, 편집할 수 있다
 - 로컬의 마크다운 문서를 불러오기 할 수 있다
    - 파일 탐색기처럼 폴더로 이동해서 열기
    - 로컬 내 파일 위치 경로를 복사해서 붙여넣기하면 바로 열기 (마치, 로컬 HTML파일을 위치 저장해서 Chrome 주소창에 붙여넣기하면 열리는 것처럼)
 - 열려 있는 마크다운 문서를 편집하고 저장할 수 있다
 - 열려 있는 마크다운 문서 행(Line) 하나를 선택하거나, 드래그하면 여러 행을 선택할 수 있다.
 - 행을 선택하면 작은 박스가 뜬다. 그 박스에 메모를 하면 우측 레이아웃에 해당 문서 행이 표기, 참조가 되면서 그 밑에 자유롭게 메모를 할 수 있다
 - 문서 하나에 대해서 피드백할 부분을 하나하나 정리하면 한큐에 해당 문서에 각 열의 메모가 우측 레이아웃에 표기
 - 기본 레이아웃은 1:1 비율
    - 좌측 레이아웃 : 기존 마크다운 문서
    - 우측 레이아웃 : 좌측 기존 마크다운 문서를 중심으로 한 메모장 (이후 따로 저장할 수 있고, 클립보드 복사만 할 수 있음)
 - 핵심은 마크다운 문서 Line별로 메모를 빠르게 기입해서 최초 AI가 생성한 문서에 대한 위치별 피드백을 한번에 할 수 있다


## 3. 완료 기준
 - 참조 문서(좌측 레이아웃) 처음부터 끝까지 읽고 수정, 보완, 피드백해야 할 부분을 한번에 기입할 수 있다

## 4. 금지 / 예외
 - AI, LLM API 연동한 에이전트 창 구현하지 않는다(예 : Cursor 에이전트 대화창 등)
 - 미리보기 기능

## 5. 맥락
 - 최대한 가벼운 기술 스택 활용
 - 디자인은 White & Black 깔끔한 스타일 
 - Vercel, Railways에 배포하면 트래픽은 크게 안 나오게 (나 혼자 쓸 예정)
