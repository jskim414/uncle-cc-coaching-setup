# 엑셀 위젯 — 구현 계획

## 한줄 요약
엑셀 파일을 읽어 항상-최상위(Always on Top) 플로팅 위젯으로 표시하고, 셀 클릭 한 번으로 클립보드 복사가 되는 Windows 데스크톱 앱.

---

## 1. 기술 스택

| 항목 | 선택 | 이유 |
|------|------|------|
| 언어 | Python 3.12+ | 빠른 개발, openpyxl 생태계 |
| UI | **tkinter** | 별도 설치 없음, 가벼움, always-on-top 지원 |
| 엑셀 읽기 | openpyxl | .xlsx 표준 라이브러리 |
| 클립보드 | pyperclip | 크로스 플랫폼 클립보드 복사 |
| 배포 | PyInstaller | 단일 .exe 생성 → 비개발자도 사용 가능 |

> **[임의 결정]** tkinter 선택 — PyQt는 더 예쁘지만 라이선스/용량 오버헤드. 가벼운 유틸리티에는 tkinter가 적합.

---

## 2. 핵심 기능

### 2-1. 파일 열기
- 실행 시 파일 선택 다이얼로그 (또는 드래그앤드롭)
- .xlsx 파일 로드
- 시트가 여러 개면 시트 선택 드롭다운 표시

### 2-2. 데이터 표시
- 스프레드시트를 테이블(Treeview) 형태로 위젯에 표시
- 열 헤더 표시 (첫 행을 헤더로 사용)
- 스크롤 지원 (세로/가로)

### 2-3. 클릭-복사
- **셀 클릭** → 해당 셀 값이 클립보드에 즉시 복사
- 복사 완료 시 시각적 피드백 (셀 배경 깜빡임 또는 토스트)

### 2-4. Always on Top
- 윈도우 항상 최상위 (`topmost` 속성)
- 토글 버튼으로 on/off 전환 가능
- 타이틀바 최소화 (컴팩트 모드)

### 2-5. 위젯 조작
- 드래그로 위치 이동
- 리사이즈 가능
- 투명도 조절 슬라이더 (작업 방해 최소화)

## 3. UI 레이아웃

```
┌─────────────────────────────────────┐
│ 📌 엑셀위젯  [파일열기] [📌] [─] [×] │  ← 타이틀바 (📌=항상위 토글)
├─────────────────────────────────────┤
│ 시트: [Sheet1 ▼]    투명도: [===●]  │  ← 컨트롤바
├──────┬──────┬──────┬────────────────┤
│ 이름 │ 은행 │ 계좌 │ 주민번호       │  ← 헤더
├──────┼──────┼──────┼────────────────┤
│ 홍길동│ 국민 │ 1234 │ 900101-1******│  ← 클릭하면 복사
│ 김철수│ 신한 │ 5678 │ 850202-1******│
│  ...  │      │      │               │
└──────┴──────┴──────┴────────────────┘
         💬 "계좌번호 복사됨!"           ← 토스트 알림
```

## 4. 파일 구조

```
excel-widget/
├── main.py              # 엔트리포인트
├── widget/
│   ├── __init__.py
│   ├── app.py           # 메인 윈도우 (tkinter)
│   ├── table.py         # 테이블 뷰 (Treeview)
│   ├── toolbar.py       # 상단 컨트롤바
│   └── toast.py         # 복사 피드백 토스트
├── core/
│   ├── __init__.py
│   ├── excel_reader.py  # openpyxl 래퍼
│   └── clipboard.py     # 클립보드 유틸
├── assets/
│   └── icon.ico         # 앱 아이콘
├── requirements.txt
├── build.bat            # PyInstaller 빌드 스크립트
└── README.md
```

> **[임의 결정]** 모듈 분리 구조 — 단일 파일로도 가능하지만, 유지보수성을 위해 분리.

## 5. 구현 순서

### Phase 1: 코어 (MVP)
1. `excel_reader.py` — openpyxl로 .xlsx 읽기
2. `app.py` — tkinter 윈도우 생성 + always-on-top
3. `table.py` — Treeview로 데이터 표시
4. 셀 클릭 → 클립보드 복사
5. 복사 피드백 (셀 하이라이트)

### Phase 2: UX 개선
6. 시트 선택 드롭다운
7. 투명도 조절
8. 위젯 위치/크기 기억 (다음 실행 시 복원)
9. 토스트 알림

### Phase 3: 배포
10. PyInstaller로 .exe 빌드
11. 아이콘 추가
