# 리서치 계획 수립 요청 프롬프트

## 기본 맥락

### Topic:

```
리서치하려는 핵심 주제

- 예
 - AI 기반 블로그 작성 SaaS 시장 분석
 - 초기 스타트업 IR 코칭 서비스 시장 구조 분석
```

### Context:

```
이 리서치를 하는 이유
어떤 의사결정을 위해 필요한지 설명

- 예
 - 새로운 SaaS 아이디어 검증을 위해 시장 구조 파악 필요
 - 경쟁 서비스 포지셔닝 파악 목적
 - 콘텐츠 전략 수립을 위한 인사이트 확보
```

### My Hypothesis:

```
현재 내가 알고 있는 가설 / 배경지식

- 예
- AI 블로그 도구는 많지만 대부분 SEO 중심
 - 실제 블로거 workflow를 깊이 이해한 제품은 적어 보임
 - ChatGPT 기반 워크플로우가 대체 가능할 가능성 있음
```

### Key Questions:

```
리서치를 통해 알고 싶은 핵심 질문

- 예
 - 주요 플레이어는 누구인가
 - 제품 차별화 포인트는 무엇인가
 - 가격 모델은 어떻게 구성되어 있는가
 - 시장 성장률 및 트렌드는 무엇인가
```

### Scope:

```
리서치 범위

- Geography
 - Global
 - Korea
 - 특정 국가

- Industry
 - SaaS
 - Creator tools
 - Enterprise software

- Time Range
 - 최근 3년
 - 최근 5년
 - 현재 기준
```

### Desired Output:

```
리서치 결과에서 뽑고 싶은 핵심 정보

- 예
 - 시장 구조
 - 주요 경쟁사
 - 비즈니스 모델 패턴
 - 가격 전략
 - 성공/실패 사례
 - 고객 세그먼트
```

### Aplication:

```
리서치 결과를 어디에 쓸 것인지

- 예
 - 유튜브 콘텐츠 기획
 - 제품 기획
 - 투자 전략 수립
 - 블로그 아티클 작성
```


### 기타

#### Output Format
```
결과물 형식
```
예
 - Markdown 리포트
 - 경쟁사 비교 표
 - 핵심 인사이트 요약
 - 전략 시사점
예시 구조
 - Market Overview
 - Key Players
 - Business Model Patterns
 - Key Insights
 - Strategic Implications

#### Depth Level

```리서치 깊이```
 - Level 1 : 개요 조사 (빠른 탐색)
 - Level 2 : 구조 분석
 - Level 3 : 전문가 수준 딥리서치

9. Additional Instructions

```추가 요청```
예
 - 실제 사례 위주
 - 숫자 데이터 포함
 - 마케팅 전략 중심 분석
